# Historial de Cambios (Changelog)

Todas las mejoras notables de **Radio Satelital** se documentarán en este archivo.

## [v9.5] - 2026-02-05
### Añadido
- **Detección Agresiva de País:** Nueva lógica en `main.js` para detectar región (Perú, España, México, etc.) y sintonizar automáticamente.
- **Sistema de Caché:** `Cache Buster` implementado en JS y CSS para asegurar que los usuarios siempre vean la última versión.
- **Modo Wear:** Nuevos temas visuales inspirados en wearables (Ocean, Sunset, Galaxy).

### Mejorado
- Optimización del Service Worker para reproducción en segundo plano.
- Corrección de estilos en el menú lateral.

## [v8.5] - 2026-01-28
### Añadido
- Soporte oficial para TWA (Trusted Web Activity).
- Archivo `assetlinks.json` para validación en Android.
- Nuevos badges de países en la lista de emisoras.
